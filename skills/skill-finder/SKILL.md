# Skill Finder

## When to Use

- The user asks to find, import, copy, adapt, or catalog more skills.
- A registry needs source attribution and quality filtering.
- Public skills should be localized for TeamAligned workflows.

## Workflow

1. Search official and high-signal repositories first.
2. Record source URL, license context, and the reason for inclusion.
3. Prefer concise adapted skills over verbatim copies unless the license and user request support copying.
4. Validate manifest, entry file, tags, and catalog count.
5. Rebuild catalog and report what was added.

## Selection Criteria

- Clear trigger condition.
- Practical workflow and verification guidance.
- Low maintenance burden.
- Useful across TeamAligned projects.
