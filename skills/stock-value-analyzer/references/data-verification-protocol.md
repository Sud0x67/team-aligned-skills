# 多信源数据校验协议

用于 `stock-value-analyzer` 启动分析前和交卷前复验。目标是防止用错价格、错口径财务数据或过时事件导致结论反转。

## 必校验字段

| 字段 | 最低信源 | 重点陷阱 |
|------|----------|----------|
| 当前股价、市值 | 3 | 现价、收盘价、目标价、52 周区间、DCF 估值混淆 |
| PE-TTM、估值分位 | 2 | TTM/静态 PE、GAAP/经调整、分位周期不同 |
| 营收、归母净利 | 2 | 财年错位、合并/母公司口径、一次性损益 |
| 毛利率、净利率、ROE | 2 | 分部/整体、加权/摊薄、扣非/全口径 |
| 分红、派息率、股息率 | 2 | 中期+年末、特别股息、回购是否计入 |
| 负债、现金流 | 2 | 有息/无息、租赁负债、FCF 定义差异 |
| 股东结构、行业地位 | 2 | 一致行动人、信托穿透、市占率口径 |

## 价格数据红线

价格类数据必须回答 5 问：

1. 这是当前成交价或最新收盘价吗？
2. 是否标明取数日期、时间、交易所时区？
3. 是否标明币种？
4. 是否有 3 个独立权威源，且差异通常不超过 2%？
5. AH/ADR 等跨市场标的是否分别独立取数？

任一答案为否，停止分析并重新取数。

## 信源优先级

- A 级：公司公告、年报、季报、业绩会材料、交易所披露、监管文件。
- B 级：Wind、Choice、理杏仁、东方财富、雪球、富途、Bloomberg、Refinitiv、Capital IQ。
- C 级：券商研报、Reuters、FT、WSJ、财新、行业协会报告。
- D 级：社交媒体、股吧、自媒体，只能作情绪参考。

关键财务数据至少需要 1 个 A 级或可追溯到 A 级的来源。

## 数据冲突处理

- 差异 ≤ 5%：可取更权威来源或均值，并备注差异。
- 差异 5%-15%：引入第三源，以 A 级来源优先。
- 差异 > 15%：追溯公告原文；无法确认则降权并标注数据存疑。
- 差异 > 30%：暂停正式结论，先报告数据冲突。

## 校验记录模板

```markdown
| 字段 | 数据类型 | 采用值 | 信源 1 | 信源 2 | 信源 3 | 取数时间 | 差异说明 |
|------|----------|--------|--------|--------|--------|----------|----------|
```
