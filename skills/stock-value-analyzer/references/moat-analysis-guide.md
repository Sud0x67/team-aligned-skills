# 护城河分析指南

护城河不是一句“龙头”或“品牌强”，必须能解释为什么利润不会被竞争者侵蚀。

## 常见护城河

| 类型 | 证据 | 反证 |
|------|------|------|
| 品牌 | 溢价、复购、用户心智、长期广告效率 | 只能靠促销、年轻用户流失 |
| 成本 | 单位成本低、规模采购、独占资源 | 成本优势被新技术或新产能抹平 |
| 转换成本 | 客户迁移成本高、数据/流程绑定 | 合同短、替代品迁移容易 |
| 网络效应 | 用户越多价值越高 | 多平台共存、补贴驱动 |
| 牌照/专利 | 稀缺授权、专利保护、监管门槛 | 牌照开放、专利到期、绕开技术 |

## 评分步骤

1. 先列出候选护城河类型。
2. 为每种类型找硬证据：价格、份额、毛利率、留存、客户集中度、专利或牌照文件。
3. 检查过去 3-5 年护城河是在加深、稳定还是被侵蚀。
4. 识别最强反证，不要只写支持材料。
