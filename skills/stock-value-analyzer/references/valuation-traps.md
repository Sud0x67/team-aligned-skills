# 估值陷阱检查清单

低估值不等于便宜。先解释估值为什么低，再判断市场是否错杀。

## 常见陷阱

| 陷阱 | 典型表现 | 检查方法 |
|------|----------|----------|
| 盈利下滑 | PE 低但利润趋势恶化 | 看收入、毛利率、订单、现金流 |
| 周期顶部 | 静态利润很高，下一轮回落 | 看行业周期、库存、价格趋势 |
| 一次性利润 | 非经常性收益撑高净利 | 拆 GAAP、扣非、经调整利润 |
| 高杠杆 | 股权便宜但债务风险高 | 看有息负债、利息覆盖、到期结构 |
| 价值毁灭 | 低 PB 但资产质量差 | 看减值、ROE、现金回收 |
| 股息陷阱 | 高股息来自不可持续分红 | 看 FCF、派息率、特别股息 |

## 必答问题

1. 估值低是因为市场情绪，还是基本面恶化？
2. 当前利润是否可持续？
3. 现金流是否支持利润和分红？
4. 行业是否正在被替代或监管重塑？
5. 如果利润下降 30%，估值还便宜吗？

## 结论分级

- 通过：估值低但基本面和现金流仍可验证。
- 存疑：有便宜特征，但关键数据或趋势冲突。
- 确认陷阱：低估值主要来自持续恶化或不可持续利润。
