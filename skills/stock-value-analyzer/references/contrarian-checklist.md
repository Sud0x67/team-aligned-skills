# 逆向投资检查清单

逆向投资不是和市场反着来，而是在市场过度悲观时确认基本面、价格和风险补偿是否同时成立。

## 三重验证

| 验证 | 通过条件 |
|------|----------|
| 行业还好吗 | 行业需求和盈利池未被永久破坏 |
| 公司还好吗 | 护城河、现金流、资产负债表仍可支撑恢复 |
| 价格够便宜吗 | 估值和安全边际足以补偿不确定性 |

三项缺一不可。只便宜但行业或公司坏掉，不是逆向机会。

## 情绪信号

- 新闻和研报叙事高度一致地悲观。
- 估值分位接近历史低位。
- 短期事件被长期线性外推。
- 机构持仓或成交极度冷淡。

情绪信号只能说明可能有机会，不能替代基本面验证。

## 反向压力问题

1. 市场担心的核心问题是否真实存在？
2. 如果最悲观假设兑现，公司还能活下来吗？
3. 现金流和资产负债表能撑过多久？
4. 当前价格是否已经反映坏消息？
5. 有什么证据能证明市场正在错杀，而不是自己看漏了？

## 输出建议

- 当前市场情绪
- 市场担心什么
- 哪些担心合理
- 哪些担心可能被高估
- 逆向机会是否成立
- 需要跟踪的触发条件
