# 三好原则评分参考

核心公式：

```text
投资价值 = 好行业 x 好公司 x 好价格
```

## 好行业

| 维度 | 高分特征 | 低分特征 |
|------|----------|----------|
| 行业格局 | 寡头、集中度提升、价格战少 | 分散、竞争激烈、价格战频繁 |
| 定价权 | 能提价且销量不明显受损 | 成本上涨无法传导 |
| 需求稳定 | 刚需、复购、长期存在 | 风口、强周期、易替代 |
| 进入壁垒 | 牌照、品牌、网络效应、规模、技术 | 低资本门槛、低转换成本 |

## 好公司

- 护城河：品牌、成本、转换成本、网络效应、牌照/专利。
- 盈利能力：毛利率、净利率、ROE、ROIC 是否长期高于同行。
- 财务健康：有息负债、现金流、应收账款、存货、商誉。
- 管理层：诚信、资本配置、股东回报、战略稳定性。

## 好价格

先问基本面，再看估值。PE 低可能是机会，也可能是市场预期盈利下滑。

检查顺序：

1. 收入、利润、现金流是否仍健康。
2. 毛利率和竞争地位是否恶化。
3. 当前估值与历史、同行、增长质量是否匹配。
4. 安全边际是否足以覆盖核心不确定性。

## 一票否决

- 行业长期衰退且无转型证据。
- 公司没有可识别护城河。
- 财务造假、重大违规或管理层诚信破产。
- 连续多年经营现金流为负且解释不足。
- 高杠杆且没有稳定现金流覆盖。
